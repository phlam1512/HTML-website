About

The purpose of this website is to promote and advocate the idea of discrimination and stereotyping ceasing from the society through introducing many real-life examples and explain the effect and impact of discrimination and stereotyping on the so called “criminals” and “victims”, as well as the society as a whole. Moreover, this project of creating and programming a website using HTML5 and CSS, about discrimination and stereotypes, will be a great opportunity for me to not only learn more about these concepts, but also to learn and develop skills on website programming.
